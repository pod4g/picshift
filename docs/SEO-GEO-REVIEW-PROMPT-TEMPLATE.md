# SEO / GEO 复盘提示模板

下次需要复盘本轮 SEO / GEO 优化效果时，可以直接把下面这段发给 AI，然后再附上最新的 GSC 数据或截图。

## 短模板

```text
先看 `docs/SEO-GEO-SPRINT-SUMMARY-2026-03-07.md`
再看 `docs/GSC-QUERY-ACTION-MAP-2026-03-07.md`

现在我给你本次优化后的最新 GSC 数据，请你基于“2026-03-07 这轮 SEO / GEO 冲刺”做复盘分析：

1. 判断这轮优化整体是否正向
2. 对比优化前后的曝光、排名、CTR、页面承接关系
3. 找出哪些页面明显变好了，哪些页面没有变化，哪些页面变差了
4. 判断本轮优化更偏向带来了“收录改善”“相关性改善”还是“点击改善”
5. 给出下一轮最值得做的 3-5 个动作

请不要从零开始泛泛讲 SEO，而是基于这两份文档和我给你的 GSC 数据，做对比分析。
```

## 强化版模板

如果你这次给的数据比较全，可以用这版：

```text
先看 `docs/SEO-GEO-SPRINT-SUMMARY-2026-03-07.md`
再看 `docs/GSC-QUERY-ACTION-MAP-2026-03-07.md`

下面是我这次提供的 GSC 数据：
- 最近 7 天 vs 前 7 天
- 最近 14 天 vs 前 14 天
- 最近 28 天 vs 优化前 28 天
- 查询数.csv
- 网页.csv
- 未收录 / 已发现未收录截图

请你基于这些数据，复盘 2026-03-07 这轮 SEO / GEO 优化的效果，并按下面结构输出：

1. 总结：这轮优化整体是正向、无明显变化、还是部分负向
2. 重点发现：按影响大小列出最重要的 5 条结论
3. 页面分析：哪些重点页变好，哪些没起量，哪些需要重做
4. Query 分析：哪些 query 开始集中到正确页面，哪些还在分散或错配
5. 收录分析：这轮优化是否改善了抓取 / 收录优先级
6. GEO 分析：这轮优化是否提升了页面的可理解性、可引用性、证据链完整度
7. 下一轮动作：只给我最值得做的 3-5 个动作，按优先级排序

注意：
- 不要脱离文档另起炉灶
- 不要泛泛讲 SEO 常识
- 重点做“前后对比”
- 如果证据不足，请明确说“暂时无法下结论”
```

## 极简版

如果你只是想快速判断方向，可以只发这版：

```text
先看 `docs/SEO-GEO-SPRINT-SUMMARY-2026-03-07.md`
再看 `docs/GSC-QUERY-ACTION-MAP-2026-03-07.md`
然后结合我现在给你的 GSC 数据，判断这轮 SEO / GEO 优化是否正向，并告诉我下一步最值得做的 3 个动作。
```

## 推荐做法

- 7 天后先用“短模板”看方向
- 14 天后用“强化版模板”看是否形成稳定信号
- 28 天后再做一次完整结论
